Fraud Detection Using Julia

This project is a small fraud detection demonstration built around a Julia backend and a simple HTML interface. The system reads transaction data from a PaySim-style dataset, evaluates each transaction using a rule-based scoring approach, and then classifies it as low, medium, or high risk based on the patterns it finds.
